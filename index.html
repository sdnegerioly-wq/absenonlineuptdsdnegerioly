📱 QR Code Absensi Sekolah

Cetak QR Code ini dan pasang di depan kelas/sekolah. Guru scan QR Code untuk absen dengan verifikasi lokasi GPS otomatis.

UPTD SD Negeri Oly

📍 Pengaturan Lokasi Sekolah

✅ QR Code Permanen: QR Code ini sama di semua perangkat (HP & komputer). Cukup cetak sekali dan tempel di sekolah.

📍 Cara Kerja:
1. Admin cetak QR Code → tempel di sekolah
2. Guru buka aplikasi di HP → klik "Scan QR Code"
3. Sistem cek GPS → harus dalam radius 100 meter dari sekolah
4. Jika lokasi valid → absen otomatis tercatat ✅
5. Jika di luar sekolah → absen ditolak ❌
